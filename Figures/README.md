# Figures

This directory contains aggregate preview figures for the TCM-Evaluation manuscript repository.

The figures are included to make the GitHub project page informative during review. They do not contain patient-level clinical text, individual case descriptions, physician reports, model outputs, or expert scoring records.

Included preview figures:

- `per_case_score_distributions.png`: per-case expert score distributions for models and the human comparator across nine evaluation dimensions.
- `inter_rater_agreement_heatmaps.png`: pairwise expert-rater agreement across scoring dimensions.

Complete source data and final manuscript figures will be released after manuscript acceptance.
